# ReactJS Learning Project

This project is a React learning project built with Vite + React and styled using TailwindCSS.
It covers React fundamentals and concepts, including:

Component creation and lifecycle methods

State and props management

React Hooks (useState, useEffect, useContext, etc.)

Component composition and reusable components

Global state management (Context, Redux/MobX)

Forms, validation, and form libraries (Formik/React Hook Form)

Styling with TailwindCSS and styled-components

Custom hooks for reusable logic

 #Installation

Clone this repository and install dependencies:

git clone https://github.com/kevintank99/react-intermediate.git
cd react-intermediate
npm install

To start the development server:

npm run dev
